Computer Processor

I developed a single-core and multi-core computer processor capable of running basic programs.

I bagan by implementing both a fixed-latency and variable-latency iterative integer multiplier. The datapath for the variable-latency multiplier is shown below.


Next, I implemented both a stalling and a fully bypassed pipelined processor. This processor uses the previously implemented multiplier. The datapath for the fully bypassed processor is shown below.


I then implemented both a direct-mapped and a 2-way-set-associative cache. The datapath for the set-associative cache is shown below.


Finally, I put everything together and created a single-core and multi-core processor. The single-core processor is made up of an instruction cache, data cache, and processor. The multicore system is made up of four private instruction caches, a four-banked shared data cache, four processors, and four ring networks.
