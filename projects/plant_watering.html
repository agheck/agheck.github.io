Automatic Plant Watering System

I developed a system capable of automatically watering a plant. The system monitors the plant's soil moisture, the air temperature and humidity, and the ambient light value. It displays these values on the home screen, along with the time that the plant was last watered. The user may select from 3 watering modes: manual, moisture sensing, and interval.

Manual Mode

The user must click the on/off button to start and stop the water. There is no automatic watering in this mode.

Moisture Sensing Mode

The system first calculates the target soil moisture value. It begins by prompting the user for their plant type, then their specific plant. If the user's specific plant is in the database, the system uses that value as the target. Otherwise, the system uses the average value of all of the plants in the category as the target. The user may also specify their own target value.

Once the target value is selected, the system continuously checks the soil moisture level. If the moisture level is below the target value, the water pump is turned on until the moisture level is at the target value.

Interval Mode

The user selects the frequency and duration at which they want the plant watered. The system automatically turns the water on and off at the specified frequency and duration.
