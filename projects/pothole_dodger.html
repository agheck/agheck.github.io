Pothole Dodger

I developed a game inpsired by Crazy Taxi. The game is displayed in a Python GUI created by Tkinter. The user must avoid potholes in a 3-lane road by switching lanes. The user may switch lanes by pressing buttons and may start/pause/resume the game by titling the board up or down. The game ends when the player hits a pothole.
